# Техническое задание: Персональный сайт-портфолио

## Цель

Создать современный одностраничный сайт-портфолио для презентации моих навыков, услуг и выполненных проектов. Сайт должен выглядеть профессионально, вызывать доверие и помогать потенциальным клиентам быстро понять, чем я занимаюсь и как со мной связаться.

## Общие требования

- Одностраничный сайт (Landing Page).
- Адаптивная верстка для компьютеров, планшетов и смартфонов.
- Современный минималистичный дизайн.
- Светлая цветовая схема с аккуратными акцентами.
- Плавная прокрутка между блоками.
- Чистый и хорошо структурированный код.
- Использовать React + TypeScript + Tailwind CSS.
- Все тексты должны быть вынесены в отдельный объект или файл для удобного редактирования.

# Структура сайта

## 1. Hero-блок (Первый экран)

Содержимое:

- Имя и фамилия.
- Краткое позиционирование.
- Небольшое описание в 2–3 предложениях.
- Фото или аватар.
- Кнопка «Связаться».
- Кнопка «Посмотреть работы».

## 2. Обо мне

- Краткая история обо мне.
- Чем я занимаюсь.
- Какие задачи помогаю решать клиентам.
- Почему со мной удобно работать.

Преимущества:

- Быстрая реализация идей.
- Понятная коммуникация.
- Современные технологии.
- Индивидуальный подход.

## 3. Навыки

### Разработка

- HTML
- CSS
- JavaScript
- TypeScript
- React
- Next.js

### AI-инструменты

- Cursor
- ChatGPT
- Claude
- Генерация контента
- Автоматизация задач

### Дополнительно

- UX/UI мышление
- Создание MVP
- Прототипирование
- Работа с API

## 4. Для кого я могу быть полезна

Карточки:

- Предпринимателям
- Экспертам
- Малому бизнесу
- Стартапам

## 5. Портфолио

Для каждой работы отображать:

- Изображение проекта.
- Название.
- Краткое описание.
- Использованные технологии.
- Ссылку на демо.
- Ссылку на GitHub.

Подготовить минимум 6 карточек-заглушек.

## 6. Этапы работы

1. Обсуждение задачи.
2. Подготовка решения.
3. Разработка.
4. Тестирование.
5. Запуск проекта.

## 7. Контакты

Добавить:

- Email
- Telegram
- LinkedIn
- GitHub

Форма обратной связи:

- Имя
- Email
- Сообщение

После отправки показывать уведомление об успешной отправке.

## 8. Footer

- Имя
- Год
- Ссылки на социальные сети
- Кнопка возврата наверх

## Дополнительные требования

- Плавные анимации появления блоков.
- Современные UI-компоненты.
- Хорошие отступы между секциями.
- SEO-метатеги.
- Оптимизация производительности.
- Возможность легко менять тексты, ссылки и проекты.

## Результат

Полностью готовый адаптивный сайт-портфолио с современным дизайном, готовый к деплою на Vercel.
