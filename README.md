# Портфолио — Людмила Киселева

Одностраничный сайт-портфолио веб-разработчика. Работает без Node.js и сборки — достаточно открыть `index.html` в браузере.

## Быстрый старт

1. Откройте файл `index.html` двойным щелчком или перетащите его в окно браузера.
2. Для просмотра с телефона можно загрузить папку на любой хостинг статики.

Node.js, npm и установка зависимостей **не требуются**.

## Структура проекта

```
portfolio/
├── index.html          # Главная страница
├── css/
│   └── style.css       # Стили и анимации
├── js/
│   ├── content.js      # Все тексты, ссылки и данные
│   └── main.js         # Отрисовка страницы и логика
├── favicon.svg         # Иконка сайта
├── avatar.jpg          # Ваше фото (добавьте сами, необязательно)
├── portfolio-site-tz.md
└── README.md
```

## Секции сайта

- **Hero** — имя, описание, кнопки, фото
- **Обо мне** — история и преимущества
- **Навыки** — технологии и инструменты
- **Для кого** — целевая аудитория
- **Портфолио** — карточки проектов
- **Этапы работы** — процесс сотрудничества
- **Отзывы** — отзывы клиентов
- **Контакты** — ссылки и форма обратной связи

## Редактирование контента

Все тексты, контакты, проекты и отзывы находятся в одном файле:

**`js/content.js`**

Примеры того, что можно менять:

| Блок | Что редактировать |
|------|-------------------|
| `hero` | Имя, фамилия, описание, текст кнопок |
| `about` | Тексты «Обо мне», преимущества |
| `skills` | Категории и список навыков |
| `portfolio.projects` | Названия, описания, технологии, ссылки |
| `reviews.items` | Имена, должности, текст отзывов |
| `contact.links` | Email, Telegram, Max, GitHub |
| `footer` | Имя и подпись в подвале |

После изменений сохраните файл и обновите страницу в браузере (F5).

## Фото

Положите файл **`avatar.jpg`** в корень папки `portfolio`. Если фото нет, отображается стильная заглушка.

## Контакты (заглушки)

Замените значения-заглушки в `contact.links` на свои:

```js
{ type: 'email', label: 'Email', value: 'hello@example.com', href: 'mailto:hello@example.com' },
{ type: 'telegram', label: 'Telegram', value: '@username', href: 'https://t.me/username' },
{ type: 'max', label: 'Max', value: '@username', href: 'https://max.ru/username' },
{ type: 'github', label: 'GitHub', value: 'github.com/username', href: 'https://github.com/username' },
```

## Публикация в интернет

Сайт — набор статических файлов. Подойдут:

- [GitHub Pages](https://pages.github.com)
- [Netlify Drop](https://app.netlify.com/drop) — перетащите папку
- [Vercel](https://vercel.com) — загрузите папку как статический сайт
- Любой другой хостинг с поддержкой HTML

## Дизайн

- Графитовый фон
- Акценты: красный, белый, жёлтый
- Шрифты: Syne (заголовки), Outfit (текст) — подгружаются из Google Fonts (нужен интернет)
- Адаптивная вёрстка для телефонов, планшетов и компьютеров

## Форма обратной связи

Форма показывает сообщение об успешной отправке на стороне браузера. Для реальной отправки писем нужно подключить сервис (Formspree, Netlify Forms и т.п.) — это отдельная настройка.
